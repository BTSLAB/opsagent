# Contributing to OpsAgent

Welcome to the OpsAgent project by BTS Labs!

## How to Contribute
1. **Bugs & small fixes** - Open a PR!
2. **New features / architecture** - Start a GitHub Discussion or issue first
3. **Questions** - Open a GitHub issue

## Before You PR
- Test locally with your OpsAgent instance
- Run linter: `npm run lint`
- Keep PRs focused (one thing per PR)
- Describe what & why

## Current Focus

We are currently prioritizing:
- **Stability**: Fixing edge cases in channel connections.
- **UX**: Improving the onboarding wizard and error messages.
- **Skills**: Expanding the library of bundled skills.
- **Performance**: Optimizing token usage and compaction logic.
